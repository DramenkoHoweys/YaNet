# GUI for Cores

这个项目共开发了两个客户端软件。分别采用了 Mihomo 内核和 Singbox 内核。

<Embed>https://github.com/GUI-for-Cores/GUI.for.Clash</Embed>

<Embed>https://github.com/GUI-for-Cores/GUI.for.SingBox</Embed>

#### 特点

1. 安装包大概5M左右，太小了
2. 后端使用Go语言，运行效率很高
3. 有官方的插件平台
4. 支持覆写
5. 有定时任务功能
