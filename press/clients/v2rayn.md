# v2rayN

适用于 Windows 的图形用户界面客户端，支持 Xray core 和 v2fly core 等。

<Embed>https://github.com/2dust/v2rayN</Embed>

#### 特点

1. C# 编写，非常适合 Windows 平台
2. 系统资源占用比较低
